## 4. 제어구문

> * 순차형
>
> * 선택형
>
>   * 조건문 if, switch
>
>     ```java
>     if(조건문) {
>         // 조건이 만족할 때 실행할 명령문
>     } else {
>         // 조건이 만족하지 않을 때 실행할 명령문
>     }
>     switch(조건식) {
>             // 판단할 수 있는 값
>             // 변수, 연산식, 메소드 호출문
>             //   -> int로 casting 할 수 있는 타입 + String
>     }
>     ```
>
>     * 제어구문은 중첩해서 사용하는 경우가 많으므로 반드시 `들여쓰기`를  잘 맞추어서 사용한다. 
>     * if문은 조건식의 결과가 true, false두 가지 밖에 없기 때문에 경우의 수가 많아질수록 복잡해지고 처리 시간이 오래 걸린다.
>
> * 순환형
>
>   * 반복문 for, while, do - while
